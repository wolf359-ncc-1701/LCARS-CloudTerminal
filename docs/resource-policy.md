# Resource And Copyright Policy

This project studies meWho/Titan.DS as a reference for interaction design, animation structure, and high-density interface composition.

It should not redistribute meWho/Titan.DS assets or Star Trek copyrighted/trademarked materials.

## Do Not Commit

- Downloaded Titan.DS JS chunks.
- Titan.DS MP3 sound files.
- Titan.DS WebP/PNG/SVG art.
- Star Trek logos, ship diagrams, faction emblems, or official LCARS graphics.
- Screenshots from the reference site except as private local notes.

## Allowed Direction

- Original CSS shapes inspired by general LCARS/FUI conventions.
- Original SVG/CSS icons.
- Generated or hand-authored abstract diagrams.
- Web Audio API synthesized beeps and alerts.
- Open-license fonts and assets with attribution as required.
- User-provided fonts may be committed only when tracked in `ASSETS.md`; confirm redistribution rights before public release packaging.
- User-owned screenshots or notes kept outside release assets.

## If External Assets Are Needed

Track each asset in a future `ASSETS.md` with:

- File path.
- Source URL.
- License.
- Attribution requirement.
- Date downloaded.
- Why it is needed.

## Design Boundary

The goal is an original personal command center. It can evoke:

- High-density sci-fi operations displays.
- LCARS-like rounded blocks.
- Animated telemetry.
- Modular system panels.

It should not attempt to be a public clone of Titan.DS or a direct Star Trek UI reproduction.
